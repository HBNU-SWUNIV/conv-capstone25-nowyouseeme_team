## <u>Teamate</u> Project Background
- ### 필요성
  - 공공장소에서 쓰레기 무단투기가 급연구역 내 흡연 문제까지 심화되고 있다.
  - 기존 CCTV 객체 인식 시스템은 대부분 사후 확인용으로, 실시간 감지 및 대응이 어렵다.
  - 인공지능 기반 실시간 인식 기술을 활용한 공공장소의 쾌적환경과 안전성을 향상시킬 필요가 있다.
    
- ### 기존 해결책의 문제점
  - 단순 객체 위치 기반의 기존 모델은 쓰레기 종류 분류나 흡연 동작 인식이 불가능하다.
  - 다중 객체 및 복합 행동 인식이 요구되는 실제 환경에서는 탐지 정확도와 실시간성 모두 한계가 존재한다.
  
## System Design
  - ### System Requirements
    - 쓰레기 분류(Classification): 캔, 페트병/유 비닐, 종이컵 등 쓰레기를 자동으로 인식하고 분류
    - 흡연 동작 인식(Action Recognition): 연속 프레임 기반으로 흡연 행위를 실시간 감지
    
## Case Study
  - ### Description
    - 쓰레기 및 흡연 동작 데이터셋 구축
    - YOLO 기반 탐지 모델을 설계하여 객체 탐지와 행동 인식을 동시에 수행
  
  
## Conclusion
  - ### 기술적 성과
    - 복합 패턴 인식을 동시에 수행하는 AI 원천 기술 확보
  
  - ### 사회적 효과
    - 공공장소의 환경 개선 및 간접흡연 피해 감소로 시민 안전과 건강 증진
  
## Project Outcome
- ### 2025년 공학교육혁신센터 창의종합설계경진대회 참가
